របៀបប្រើប្រាស់ Gmail "Send As"

ជំហានទី 1: ចូលទៅកាន់ Gmail

បើក Browser របស់អ្នក ហើយចូលទៅ https://mail.google.com

ជំហានទី 2: ចូលទៅកាន់ការកំណត់ (Settings)

ចុចរូប齿轮 (Gear icon) នៅខាងលើស្តាំ → ចុច "See all settings"

ជំហានទី 3: ទៅប៊ូតុង "Accounts and Import"

ក្នុងផ្ទាំង Settings → ចុច tab "Accounts and Import"

ជំហានទី 4: បន្ថែមអាសយដ្ឋានអ៊ីមែលថ្មី

រកមើលជម្រើស "Send mail as" → ចុច "Add another email address"

ជំហានទី 5: បញ្ចូលព័ត៌មានអាសយដ្ឋាន

បញ្ចូលឈ្មោះ និងអាសយដ្ឋានអ៊ីមែលដែលអ្នកចង់ផ្ញើជំនួស → ចុច Next Step

ជំហានទី 6: បញ្ចូល SMTP Server

បញ្ចូលព័ត៌មាន SMTP server របស់អ៊ីមែលដែលអ្នកបន្ថែម។ ប្រសិនបើជារាជការ សូមស្នើ IT ដើម្បីយល់ពីព័ត៌មាន Server/Port និងពាក្យសម្ងាត់។

ជំហានទី 7: Verify និងបញ្ចប់

Gmail នឹងផ្ញើលេខកូដទៅអ៊ីមែលដែលបានបន្ថែម។ ចូលអ៊ីមែលនោះ → ចម្លងលេខកូដ → បញ្ចូលវានៅក្នុង Gmail ដើម្បីបញ្ចប់។

ជំហានទី 8: ជ្រើសអាសយដ្ឋានដែលចង់ប្រើ Send As

អ្នកអាចជ្រើសឲ្យ Gmail ប្រើអ៊ីមែលថ្មីជាដើមតាម Default ឬ ជ្រើសក្នុងពេល Compose។

ជំហានទី 9: សាកល្បងផ្ញើអ៊ីមែល

ចុច Compose → ជ្រើសអាសយដ្ឋាន "From" ដែលអ្នកបានបន្ថែម → ផ្ញើទៅអ្នកទទួល ដើម្បីសាកល្បង។

⬆ ត្រឡប់ទៅលើកំពូល
